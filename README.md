## Prepare the Data(25%-30%)


**Privacy Levels** in power bi are used to control how data from different sources can be combined, preventing potential data leaks or unwanted data exposure between sources. 

Different type of Privacy levels are: 

-None (Default/Undefined): No privacy level is defined, which may trigger warnings.
-Public: Data is visible to anyone and can be freely shared or combined with any other source.
-Organizational: Data is isolated from public sources but can be shared with other trusted internal sources.
-Private: Data is restricted and isolated from all other data sources to prevent, for example, internal, secure data from leaking to a, public, web-based, dataset.
